Boca Juniors Fan Hub

Website Planning & Architecture Document

Site Name

Chosen Name: Boca Juniors Fan Hub

Description: This name directly identifies the website as a dedicated space for fans of Club Atlético Boca Juniors. It indicates that the site serves as a central "hub" where supporters can access comprehensive information regarding the club's history, current squad, matches, and achievements.

Target Domain: bocajuniorsfanhub.net (Alternative: bocafanhub.org)

Site Purpose

The purpose of this website is to provide an interactive, responsive, and informational platform for Boca Juniors supporters worldwide. The scope of the project includes:

  • Preserving and presenting the rich history and legendary achievements of the club.
  • Displaying the active first-team roster dynamically using modern JavaScript components.
  • Providing information on recent match results and news updates.
  • Engaging users via a custom interactive contact/feedback form and saving individual user preferences (like a "Favorite Player" feature) locally in the browser.

Target Audience Scenarios

To guide the layout, user experience, and architecture of the content, the site addresses the following critical user inquiries:

Scenario 1

"Who are the current active forward players in the squad, and what are their seasonal statistics?"

Site Answer: The dedicated Players Page will generate interactive player cards using a JavaScript array of objects. Users will be able to view details dynamically and save their favorite players using localStorage.

Scenario 2

"Where can I see the list of major international titles won by Boca Juniors, and how can I contact the fan club management?"

Site Answer: The historical achievements are highlighted inside structured semantic lists on the Home Page, while a standard-compliant HTML form is available on the Contact Page for direct user inquiries.

Application Rule: Deep Blue establishes the core structure (Headers, Main navigation base). Gold serves as the secondary accent color to draw focus to active elements, links, and grid borders. Dark Text over a Light/White background guarantees optimal reading clarity.

Typography

The font selection relies on modern web fonts delivered via Google Fonts to achieve a clean balance between an energetic athletic theme and standard corporate readability:

Heading Typography: 'Montserrat', sans-serif

BOCA JUNIORS: 1905 LEGENDARY GLORY

Used strictly for main titles (<h1>, <h2>, <h3>) and navigation components. Bold, geometric, and highly visible.

Body Typography: 'Open Sans', sans-serif

La Bombonera is one of the most famous football stadiums in the world. This font handles all standard paragraphs, form text, tables, and lists.
